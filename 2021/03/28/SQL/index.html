SQL

连接语句

  • r1 join r2 using (A1, A2);
//此语句代表对r1和r2进行连接，连接的条件是r1.A1 = r2.A1, r1.A2 = r2.A2
//假设就算r1,r2还有共同属性A3,对于r1.A3 != r2.A3的元组还是会连接起来
//上述 join using 语句是natural join 语句更泛化的形式


* ```sql
  //语句1
  select *
  from student left outer join takes
  on student.ID = takes.ID
  //语句2
  select *
  from student left outer join takes
  where student.ID = takes.ID

  //语句1的执行结果与上面的语句一样，除了ID会出现两次，on子句可以与外连接语句一起使用
  //语句儿执行结果不一样，对于未选课的学生不会出现在结果中，会被where子句谓词排除掉
