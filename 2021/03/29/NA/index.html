线性方程组

直接解法

消元

  • 高斯消元成上三角($\sum_{k=1}^{n-1}(n-k)(n-k+2)=n^3/3+n^2/2-5n/6$次运算)，再从最后一项回代($1+\sum_{i=1}^{n-1}(n-i+1)=n^2/2+n/2$次运算)
  • Gauss-Jordan消元：消成主对角阵

选主元

  • Partial Pivoting

    • 运行到第k行时，选取$｜a_{ik}｜$最大的一行($k<=i<=n$)并与第k行交换

  • Scaled Partial Pivoting（Scaled- Column Pivoting）

    • $s_i=max_{1<=j<=n}(|a_{ij}|)$
    • 运行到第k行时，选取$\frac{|a_{ik}|}{s_i}$最大的一行($k<=i<=n$)并与第k行交换

  • Complete Pivoting

    • 运行到第k行时，选取$|a_{ij}|$最大的元素($k<=i<=n,k<=j<=n$)，并通过行变换和列变换交换到$a_{kk}$的位置，列变换需要交换未知数的位置

矩阵和向量

  • $A_i(1<=i<=n)$是n阶方阵，则$|\prod_{i=1}^nA_i|=\prod_{i=1}^n|A_i|$
  • 对角占优矩阵
    • 每一行的对角元绝对值大于等于其余元绝对值之和
    • 若严格大于，则称为严格对角占优

迭代解法

interpolation（差值）

Lagrange Interpolation

系数

$P_n = a_0+a_1x+…+a_nx^n$

这种表示形式中$1,x,x^2,…,x^n$作为一组基

差值中假设有$P_n(x_i) = y_i(0<=i<=n)$，若用上面的一组基表示$P_n$，则需要解方程组

在Lagrange差值中，变换基为$L_{n,i}(x)$此时$P_n = \sum_{i=0}^nL_{n,i}(x)y_i$，这是为了直接让$y_i$作为系数，省去了解方程组的麻烦，$L_{n, i}$满足如下条件：

  • $x=x_i,L_{n, i}=1;x\neq x_i,L_{n,i}=0$

此时要确定Lagrange系数$L_{n,i}$

由其满足的条件可以令$L_{n,i} = c_i\prod_{j\neq i}(x-x_j)$，令$x=x_i$有$c_i\prod_{j\neq i}(x_i-x_j)=1$

所以$L_{n,i}=\prod_{j\neq i}(x-x_j)/(x_i-x_j)$

余项

$R_n(x)=f(x)-P_n(x)$至少有n+1个根，令$R_n(x)=K(x)\prod_{i=0}^n(x-x_i),a<=x_0<=…<=x_n<=b$

固定$x\neq x_i$，令$g(t)=R_n(t)-K(x)\prod_{i=0}^n(t-x_i)=(K(t)-K(x))\prod_{i=0}^n(x-x_i)$至少有$x_0,…,x_n,x$共n+2个根

所以由Rolle定理，存在$\xi_x$使得$g^{(n+1)}(\xi_x)=0$即$R^{(n+1)}_n(\xi_x)-K(x)(n+1)!=f^{(n+1)}(\xi_x)-K(x)(n+1)!=0$

故$R_n(x)=\frac{f^{(n+1)}(\xi_x)}{(n+1)!}\prod_{i=0}^n(x-x_i)$

divided differences

一阶差商：$f[x_i,x_j]=\frac{f(x_i)-f(x_j)}{x_i-x_j}$

k阶差商：$f[x_0,…,x_k]=\frac{f[x_0,…,x_{k-1}]-f[x_1,…x_k]}{x_0-x_k}=\sum_{i=0}^kf(x_i)/\prod_{j\neq i}(x_i-x_j)$

Newton’s Interpolation

$f(x)=f(x_0)+(x-x_0)f[x,x_0]$

$f[x,x_0]=f[x_0,x_1]+(x-x_1)f[x,x_0,x_1]$

…

$f[x,x_0,…,x_{n-1}]=f[x_0,…,x_n]+(x-x_n)f[x,x_0,…,x_n]$

综合上面式子

$f(x)=f(x_0)+(x-x_0)f[x_0,x_1]+(x-x_0)(x-x_1)f[x_0,x_1,x_2]+…+\prod_{i=0}^{n-1}(x-x_i)f[x_0,…,x_n]+R_n(x)$

$R_n(x)=\prod_{i=0}^n(x-x_i)f[x,x_0,…,x_n]$

此时可以得到$\frac{f^{(n+1)}(\xi_x)}{(n+1)!}\prod_{i=0}^n(x-x_i)=\prod_{i=0}^n(x-x_i)f[x,x_0,…,x_n]$

即$\frac{f^{(n+1)}(\xi_x)}{(n+1)!}=f[x,x_0,…,x_n]$

所以有如下差商公式：

$f[x_0,..,x_n]=f^n(\xi)/n!, \xi \in(x_{min},x_{max})$

  • example:

    $f(x)=2x^3+4x,x_k=kh(k=0,1,…)$

    求$f[x_1,x_2,x_3],f[x_1,x_2,x_3,x_4],f[x_1,…,x_5],f[x_1,…,x_6],f[x_1,…,x_7]$

    直接硬解比较麻烦，可以利用差商公式，f(x)求4次即以上次导就是0，所以后三个都是0

    对于第二个，求3次导后是常数12，所以$f[x_1,…,x_4]=12/3!=2$

    对于第一个只能按公式来$f[x_1,x_2,x_3]=(f[x_1,x_2]-f[x_2,x_3])/(x_1-x_3)$

    $f[x_1,x_2]=(f(h)-f(2h))/(-h)=14h^2+4$

    $f[x_2,x_3]=(f(2h)-f(3h))/(-h)=38h^2+4$

    所以$f[x_1,x_2,x_3]=12h$

Hermite Interpolation

给定$x_0,…,x_n;H_{2n+1}(x_i)=y_i,H’_{2n+1}(x_i)=y’_i$

令$H_{2n+1}(x)=\sum_{i=0}^nh_i(x)y_i+\sum_{i=0}^n\hat{h}_i(x)y’_i$

$H’_{2n+1}(x)=\sum_{i=0}^nh’_i(x)y_i+\sum_{i=0}^n\hat{h}’_i(x)y’_i$

对于$h_i(x)$，$x\neq x_i,h_i(x)=0,h’_i(x)=0;x=x_i,h_i(x)=1,h’_i(x)=0$

所以$h_i(x)=(ax+b)L^2_{n,i}(x)$

利用$h_i(x)=1,h’_i(x)=0$得到$h_i(x)=(1-2L’_{n,i}(x_i)(x-x_i))L^2_{n,i}(x)$

对于$\hat{h}_i(x),x\neq x_i,\hat{h}_i(x)=0,\hat{h}_i’(x)=0;x=x_i,\hat{h}_i(x)=0,\hat{h}_i’(x)=1$

所以$\hat{h}_i(x)=(x-x_i)L_{n,i}^2(x)$

$R_n(x)=f^{(2n+2)}(\xi_x)[\prod_{i=0}^n(x-x_i)]^2/(2n+2)!$

Cubic Spline Interpolation

给定$a=x_0<…<x_n=b$，三次样条差值S满足如下条件：

  • $S_i(x)$是三次多项式，$S_j(x)=a_j+b_j(x-x_{j-1})+c_j(x-x_{j-1})^2+d_j(x-x_{j-1})^3$
  • $S(x_i)=f(x_i),i=0,..,n$
  • $S_{i+1}(x_{i+1})=S_i(x_{i+1}),i=0,…,n-2$
  • $S’_{i+1}(x_{i+1})=S’_i(x_{i+1}),i=0,…,n-2$
  • $S’’_{i+1}(x_{i+1})=S’’_i(x_{i+1}),i=0,…,n-2$

上述条件构成n+1+3(n-1)=4n-2个方程，通过一些手段减少方程的个数

令$h_j=x_j-x_{j-1},S_j’’(x_j)=M_j$

$\lambda_i=h_{j+1}/(h_j+h_{j+1}),\mu_j=1-\lambda_j,g_j=6(f[x_j,x_{j+1}]-f[x_{j-1},x_j])/(h_j+h_{j+1})$

则有$\mu_jM_{j-1}+2M_j+\lambda_jM_{j+1}=g_j,j=1,…,n-1$

此时有n+1个未知数和n-1个方程，需再添加两个约束

  • 固定边界（Clamped boundary）

    $S’(a)=y’_0, S’(b)=y’_n,\lambda_0=\mu_n=1$

    $g_0=6(f[x_0,x_1]-y’_0)/h_1$

    $g_n=6(y’_n-f[x_{n-1},x_n])/h_n$

  • 自由边界（free boundary）

    $S’’(a)=y’’_0,S’’(b)=y’’_n,\lambda_0=\mu_n=0$

    $g_0=2y’’_0$

    $g_n=2y’’_n$

  • 周期性边界（Periodic boundary）

    如果f是周期性函数，则令$y_n=y_0$

解出$M_j$后，需求出$a_j, b_j, c_j, d_j$

$a_j=y_{j-1}$

$b_j=\frac{(y_j-y_{j-1})}{h_j}-\frac{m_{j-1}}{2}h_j-\frac{m_j-m_{j-1}}{6}h_j$

$c_j=\frac{m_{j-1}}{2}$

$d_j=\frac{m_j-m_{j-1}}{6}h_j$

Approximation Theory

最小二乘法

  • 用$P(x)$估计函数$f$使得$E_2 = \sum_{i=1}^m|P(x_i)-y_i|^2$最小

变量替换

  • $E_2$不好算的时候考虑变量替换将$P(x)$转成多项式函数

  • $y \approx P(x) = x/(ax+b)$，$Y = 1/y, X = 1/x$，问题转为：$Y \approx aX + b$

  • $y \approx P(x) = ae^{-b/x}$，$Y = lny, A = lna, X = 1/x, B = -b$，问题转为：$Y \approx A + BX$

两种形式

  • 离散形式：给点$(x_i, y_i), i=1,…,m$，使得$E = \sum_{i=1}^m|P(x_i)-y_i|^2$最小
  • 连续形式：给函数$f(x) \in D[a, b]$，使得$E = \int_a^b[P(x)-f(x)]^2dx$最小

线性无关

  • 函数集${\phi_0(x),…,\phi_n(x)}$在$[a, b]$满足：任意x，只要$\sum_{i=0}^na_i\phi_i(x)=0$，必有$a_i=0$，则该函数集线性无关
  • 若$\phi_i(x)$为i次多项式，则线性无关

广义多项式

  • 线性无关函数组的线性组合即为广义多项式

inner product and norm

  • 内积两种形式：$(f, g)$

    • $\sum_{i=1}^mw_if(x_i)g(x_i)$
    • $\int_a^bw(x)f(x)g(x)dx$

  • norm：$||f|| = \sqrt{(f, f)}$

广义最小二乘逼近

  • 寻找一个广义多项式$P(x)$使得$E = (P-y, P-y)$最小

  • 令$P(x) = \sum_{i=0}^na_i\phi_i(x)$，目标是求出$a_i$，令$\frac{\partial E}{\partial a_i} = 0$可得：$B\vec{a} = \vec{c}$

    其中$B_{ij} = (\phi_i, \phi_j), \vec{a} = (a_0,…,a_n)^T, \vec{c} = ((\phi_0, f),…,(\phi_n, f))^T$

切比雪夫多项式

问题背景

  • 给定函数$f(x)$，寻找n次多项式$P_n(x)$使得$||P-f||_{\infty}$最小，是一个minimax问题

性质

  • $f \in C[a, b]$且$f$不是n次多项式，则存在一个唯一的n次多项式$P_n(x)$使得$||P-f||_{\infty}$最小
  • Chebyshev Theorem：假设$P_n(x)$使得$||P-f||_{\infty}$最小，等价于函数$P-f$至少有n+2个最值点且最大值和最小值交替出现（且绝对值相等），这n+2个最值点组成的序列叫Chebyshev alternating sequence

问题等价转化

  • 由chebyshev theorem，$P_n(x)$是对$f$的n次差值多项式，所以$P$由n+1个差值点决定，此时转化为寻找合适的差值点问题，此时余项$|R_n(x)| = |\frac{f^{(n+1)}(\xi)}{(n+1)!}\prod_{i=0}^n(x-x_i)|$，$\xi$不可控并且大多数情况下f在n+1次求导后会十分平缓，近似为常数，故优化后面的部分
  • 令$w_n(x) = \prod_{i=0}^n(x-x_i)$，问题转化为求点列$x_i$使得$||w_n(x)||_{\infty}$最小，其中$x \in [-1, 1]$（实际应用时运用线性变换可扩展到实轴上）
  • $w_n(x) = x^n - P_{n-1}(x)$，问题进一步转化为$||x^n - P_{n-1}(x)||_{\infty}$的最小化问题，由chebyshev theorem知，$w_n(x)$在$[-1, 1]$上有交替出现的n+1个最大（小）值点

切比雪夫多项式

  • $cos(n\theta) = \sum_{k=0}^na_k(cos\theta)^k$，作变量替换$x = cos\theta \in [-1, 1]$，此时$T_n(x) = cos(n\theta) = cos(n*arc cosx)$为切比雪夫多项式

  • $T_n(x)$的n个根为$x_k = cos(\frac{2k-1}{2n}\pi),k = 1, …, n$

  • $T_0(x)=1, T_1(x)=x, T_{n+1}(x) = 2xT_n(x)-T_{n-1}(x)$，故最高次项系数为$2^{n-1}$

    • 此时易知$w_n(x) = T_n(x)/2^{n-1}$，$min||w_n(x)||_{\infty}=1/2^{n-1}$，差值点即为$T_n(x)$的根

  • $|R_n(x)| <= M/(2^n(n+1)!)$，M为$|f^{(n+1)}(x)|$的上界

example

在$[0, 1]$用多项式逼近$f(x) =e^x$，且绝对误差不超过$0.5*10^{-4}$

  • 区间变换：将[0, 1]映射到[-1, 1]

    作变量替换$t=2x-1$

  • 确定多项式阶数，注意此时处理的函数为$g(t) = e^{(t+1)/2}$

    $|R_n| <= \frac{e}{2^{n+1}}\frac{1}{(n+1)!}\frac{1}{2^n} <= 0.5*10^{-4}$，故n = 4

  • 确定差值点$t_i$，即$T_5(x)$的5个根：$cos\frac{\pi}{10}$ $cos\frac{3\pi}{10}$ $cos\frac{5\pi}{10}$ $cos\frac{7\pi}{10}$ $cos\frac{9\pi}{10}$

  • 解出差值点$x_i$，即$x_i = (t_i+1)/2$

  • 用$x_i$进行Lagrange差值

积分

Gaussian Quadrature

  • 用$\sum_{k=0}^nA_kf(x_k)$近似$\int_a^bw(x)f(x)dx$，达到2n+1阶精度，其中$x_i$为Gaussian point
  • 直接计算会解非线性方程组，考虑先将Gaussian point求出来，再求$A_i$

Theorem

  • 令$W(x)=\prod_{i=0}^n(x-x_i)$，则$x_i,i=0,..,n$为高斯点的充要条件是$W(x)$与任意次数小于n+1的多项式正交

  • proof：

    • 必要性：$x_i$为高斯点，则存在$A_i$使得$\sum_{k=0}^nA_kf(x_k)=\int_a^bw(x)f(x)dx$对任意次数小于等于2n+1的多项式都成立

      考虑$\int_a^bw(x)W(x)P_m(x)$，$P_m(x)$为m次多项式($m<=n$)

      此时$W(x)P_m(x)$为次数低于2n+1的多项式

      则有$\int_a^bw(x)W(x)P_m(x)=\sum_{i=0}^nA_iW(x_i)P_m(x_i)=0$

    • 充分性：$W(x)$与任意次数小于n+1的多项式正交，令$P_m(x)=W(x)q(x)+r(x)$，其中$P_m(x)$为m次多项式($m<=2n+1$)，$q(x)$为次数小于等于n的多项式，$r(x)$为次数小于n的多项式。

      下证存在$A_i$使得$\sum_{k=0}^nA_kf(x_k)=\int_a^bw(x)f(x)dx$对任意次数小于等于2n+1的多项式都成立

      $\int_a^bw(x)P_m(x)dx=\int_a^bw(x)W(x)q(x)dx+\int_a^bw(x)r(x)$

      因为$W(x)$与任意次数小于n+1的多项式正交，故$\int_a^bw(x)W(x)q(x)dx=0$

      即$\int_a^bw(x)P_m(x)dx=\int_a^bw(x)r(x)=\sum_{i=0}^nA_ir(x_i)=\sum_{i=0}^nA_iP_m(x_i)$
